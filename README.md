# TINETZ Home Assistant Integration

Inoffizielle Home Assistant Integration für das TINETZ Kundenportal.

## Features
- Historische Stromverbrauchsdaten
- Tägliches Update
- Mehrere Zeiträume (Tag / Monat / Jahr)
- Keine Live-Daten

## Installation (manuell)
1. Repository herunterladen
2. `custom_components/tinetz` nach `/config/custom_components/` kopieren
3. Home Assistant neu starten
4. Integration über UI hinzufügen

## Hinweis
Die Daten werden zeitverzögert (1–3 Tage) vom Netzbetreiber bereitgestellt.
